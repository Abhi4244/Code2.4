Now this is external style; for this we have to include a stylesheet instead of style tags.

Step-1 make empty notepad document and save it as "style.css"

You can give any name but its extension or at last it should end with".css"

step-2 Now make any folder and move the style sheet which you created just now to that folder.

step-3 Now move the ".html" file which you are working on (or providing the link to that css )also to that particular folder which is having that css file in it

For the code for css file go to developer tools and next to the html file csss file will appear






Style together with its value is called declaration.Ex."color:blue;"or"font-family:Algerian".
They are declarations.
    Some styles with values
  1. "color:blue|green|yellow|etc"
  2. "background-color:green|blue|orange|etc"
  3. font-family:Algerian|Monospace|Aerial|etc
  4. text-align:right|center|left
  5. font-size:any value in px or percentage.Ex:2px,5%,etc
  6. direction:ltr|rtl.

    Means left to right and right to left
  7. text-decoration:overline|underline|line-through|blink
  8. letter-spacing: value in px
  9. Word-spacing:value in px
  10. text-transform:none|capitalize|uppercase|lowercase
  11. text-shadow:h-shadow v-shadow color

    h-shadow=Horizontal shadow
    v-shadow=vertical shadow
    v and h shadow values can be in pixels
    Ex. text-shadow:3px 4px green;
  12. border:in px dotted|dashed|solid|ridge|groove|etc color

    Ex. border:groove 3px red;

There are even more style values This is your task to search them Web support is in the book at links page
